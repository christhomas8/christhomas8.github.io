    Customer Address Book

    All businesses rely on the supply and demand principle. However, customer demand varies from region to region. For a salesperson to market properly and efficiently, he or she needs to navigate potential customers based on regions. For this purpose, I developed a desktop app in Visual Studio. This Microsoft based app communicates with SQL servers and databases to search for customers by regions. It also allows for adding, modifying, or deleting customers from repositories.

    Binary Database

    The knowledge of storing data is often passed to third parties for most individuals. Therefore, I created my own database from scratch using one of the most universal languages. The purpose of this project was to learn and teach others how to store and access data. It stores information in a binary format which increases the difficulty for others or outsiders to inspect it. It runs from the command line and works great for keeping inventory!

    Artificial Neural Network

    Law enforcement has a limited amount of resources. Therefore, managing the resources properly is always necessary. Using a large amount of data from the FBI crime reports, I was able to create an artificial neural network. This neural network predicts the location, time of day, and sex of an offender. This is useful because it allows police departments to allocate their resources more efficiently. For example, where and how many officers should patrol a location or how many should respond to a call.
