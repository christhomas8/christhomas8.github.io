    Web Development Projects

    Clanora.com

    Every year, groups of people participate in Secret Santa gift exchanges. This full-stack website, hosted at: clanora.com, makes this process easy. Users can sign up by creating an account, or login if they already have an account. Users can modify their profiles, anytime they want. After a drawing takes place, group participants are assigned a unique giftee (someone to get a gift for) and are set to receive a gift from another unique user in the group. The gifters can view their assigned person's interests and information to assist with getting a more personalized and useful gift. Many other features are included in this site, such as sending emails to users, event countdowns, and more!

    Workforce Management App

    React is a popular open-source, front end, JavaScript library for building user interfaces or UI components. Using this framework, I developed a Workforce Management app for a call center UI demonstration. This app allows a user to login by communicating with a Nodejs API (which I also developed) to validate the data. If the data passes authentication, the main UI of the app is rendered and personalizes it with the user's data. This app includes other features like a randomly generated schedule, an interactive phone, a note pad, and much more!

    Web App Game

    Everyone loves games! Everyone also loves gifts! Motivated by this, I developed a personailzed Twister style Web App game. This SaaS can be used by uploading personal pictures, and pressing the spin button. The spinner stops randomly on a picture along with an instruction. That is the command that needs to be followed. The uploaded pictures also need to be printed and placed in a grid like pattern in the floor. Otherwise, this game can be played anywhere and altered to a person's preference (ex. Title). This includes game nights, special occassions, romantic dates, parties, or regular hangouts.
